Программа по автоматизации игровых процессов. Сканирует игровой процесс используя компьютерное зрение, принимает на основании некоторых функций решения. Выполняется бесконечно до остановки вручную. 

1. Сперва установить все пакеты для пипа
2. Установить Tesseract на компьютер, экзешник в папке tesseract install, не забыть поставить галочку напротив Russian в Additional language data.
3. Установить Interception через командную строку install-interception.exe /install
4. Запустить скрипт находясь в проекте для которого была создана программа.
